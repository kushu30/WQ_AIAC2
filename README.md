# AI Alphas Competition 2.0

This repository contains my submission pipeline for the WorldQuant AI Alphas Competition 2.0.

## Overview
- LLM-driven alpha generation
- Structured economic rationale
- Simulation via WorldQuant BRAIN API
- Filtering for robustness and low correlation

## Core Files
- ai_alpha_comp.ipynb: End-to-end LLM alpha generation notebook
- ace_lib.py: Simulation and Brain API utilities
- helpful_functions.py: Result processing helpers
- requirements.txt: Dependencies
